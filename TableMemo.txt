用意するテーブル
    顧客テーブル
    商品テーブル（部屋や車両の情報）
    予約テーブル
    料金テーブル
    支払いテーブル

顧客情報

    顧客ID
    氏名
    連絡先情報（電話番号、メールアドレス）
    住所
　　本人確認書類（免許証、マイナンバーなど）
    支払い情報（クレジットカードデータなど）
    過去の予約履歴


・在庫管理 : Stock
    商品ID : id
    商品カテゴリ : category
    名前 : name
    インターバル : interval（予約と予約の間隔）
    価格 : value
    個数 : num
    備考 : note

・予約管理 : Order
    予約ID : id  
    顧客ID : customer
    商品ID : product
    予約開始日時 : start
    予約終了日時 : end
    予約数 : num
    予約日時 : time
    予約状態 : state（確定、保留、キャンセルなど）
    備考 : note（ベッドタイプ、GPS必要など）

料金計算

    基本料金
    税金
    割引（会員割引など）
    追加料金（追加サービス、損害保険など）
    合計金額

決済処理

    支払いID
    予約ID
    顧客ID
    支払い金額
    支払い方法（クレジットカード、電子マネー、銀行振込なのかなど）
    支払い状態（支払い完了、未支払い、払い戻し済みなど）
